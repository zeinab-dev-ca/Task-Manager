# Task Manager App

A full-stack ASP.NET MVC web application with AI-powered task priority suggestions.

## Features
- Create, Edit, Delete tasks
- AI-powered priority suggestion using Claude AI
- Set Due Dates and Priority (Low, Medium, High)
- Mark tasks as Completed
- Built with Entity Framework Core and SQLite

## Technologies Used
- ASP.NET Core MVC (.NET 10)
- Entity Framework Core
- SQLite Database
- Bootstrap 5
- C#
- Claude AI API (Anthropic)

## How to Run
1. Clone the repository
2. Open in Visual Studio 2022
3. Add your Claude API key to appsettings.json
4. Run `Update-Database` in Package Manager Console
5. Press Ctrl+F5 to run
