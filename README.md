\# Task Manager App



A full-stack ASP.NET MVC web application for managing tasks.



\## Features

\- Create, Edit, Delete tasks

\- Set Due Dates and Priority (Low, Medium, High)

\- Mark tasks as Completed

\- Built with Entity Framework Core and SQLite



\## Technologies Used

\- ASP.NET Core MVC (.NET 10)

\- Entity Framework Core

\- SQLite Database

\- Bootstrap 5

\- C#



\## How to Run

1\. Clone the repository

2\. Open in Visual Studio 2022

3\. Run `Update-Database` in Package Manager Console

4\. Press Ctrl+F5 to run
